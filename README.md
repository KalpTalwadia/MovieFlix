# MovieFlix
A netflix clone application

## Tech Stak
* React
* Firebase
* Styled-Component
* React-Router
* React-Hooks

## [Live Project Link](https://kalptalwadia.github.io/movieflix/)

---
## Running
1. Open 'src/lib/firebase.prod.js' file and enter firebase credentials
2. Go to project directory and run:
```npm install```
3. Start The project 
```npm start```
